<h1 align="center">SquareSolver</h1>

## Description

* Solves a quadratic equation

## Install

1. clone repository

2. install files

3. write "make -B" in terminal

4. write ".\start.exe" in terminal

<p align="center"><img src="https://media.giphy.com/media/vFKqnCdLPNOKc/giphy.gif" width="50%"></p>

## Feats

* output color implemented with ANSI

* supports flags in the terminal:

    * -test_on "NameOfTestFile" => Test functions. Possible input format to the terminal ".\start.exe -test_on Tests1.txt"

    * -a Argument, -b Argument, -c Argument => Enter coefficients. Possible input format to the terminal ".\start.exe -a 1 -b 2 -c -2", equation x^2 + 2*x - 2 = 0.
